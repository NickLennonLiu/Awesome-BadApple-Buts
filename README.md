## Contributing

Pull requests are welcome. Good entries usually include:

- A working video link.
- A short description of what the video is played on or how it is rendered.
- One top-level category: `Bad Apple But` or `Other Bad Apple-related`.
- Put a video in `Bad Apple But` only when the title explicitly contains `Bad Apple` before `but`, and the video is a distinct presentation-form experiment rather than a pure music upload, compilation, or another IP fan derivative.
- Useful tags such as `format-software`, `minecraft`, `desmos`, `e-ink`, `piano`, `tesla-coil`, `stop-motion`, `fourier`, `audio-only`, or `other-ip`.

If a video is private, deleted, unrelated, duplicated, or misclassified, please open an issue or pull request.
