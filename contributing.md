# Contributing

Pull requests are welcome.

Good entries usually include:

- A working video link.
- A short description of what the video is played on or how it is rendered.
- One broad category: software/web, hardware/device, physical/visual media, audio, or meta.
- Useful tags such as `minecraft`, `desmos`, `e-ink`, `piano`, `tesla-coil`, `stop-motion`, `fourier`, `audio-only`, or `other-ip`.

This is a curated list, not a full scrape. Prefer entries with a distinct presentation surface, implementation idea, or cultural relevance.

If a video is private, deleted, unrelated, duplicated, or misclassified, please open an issue or pull request.
